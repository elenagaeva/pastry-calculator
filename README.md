# Сладкий Баланс

Бесплатный одностраничный калькулятор себестоимости кондитерских изделий для домашних кондитеров и небольших пекарен.

Весь сайт — один HTML-файл: приветственный лендинг и полноценный калькулятор в одном месте. Работает без сервера и без установки зависимостей.

## Возможности

- Расчёт стоимости ингредиентов по весу упаковки
- Учёт упаковки, декора и производственных расходов
- Оплата труда кондитера (часы × ставка)
- Наценка, округление цены и оценка прибыли
- Себестоимость партии, одной штуки и 100 г
- Тёмная и светлая тема
- Сохранение позиций упаковки/декора в `localStorage` браузера

## Быстрый старт

1. Склонируйте репозиторий или скачайте архив.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).

Дополнительная сборка не требуется.

## Как пользоваться

1. На главной странице нажмите **«Открыть калькулятор»** или **«Начать расчёт»**.
2. Заполните основную информацию об изделии.
3. Добавьте ингредиенты, упаковку и прочие расходы.
4. Укажите время работы и желаемую наценку.
5. Нажмите **«Рассчитать себестоимость»**.

Прямая ссылка на калькулятор: `index.html#calculator`

## Технологии

- HTML5, CSS3 (CSS-переменные, адаптивная вёрстка)
- Vanilla JavaScript (без фреймворков)
- `localStorage` для темы и списков расходов
